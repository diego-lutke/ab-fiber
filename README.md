# ab-fiber

A web tool to find the best fiber paths between two locations

## Tech Stack

- **Backend**: Go 1.24.4
- **Database**: SQLite
- **Frontend**: HTML, JavaScript, Tailwind CSS, Go Fiber, Alpine.js

## Installation

### Clone the Repository

```bash
git clone https://github.com/diego-lutke/ab-fiber.git
cd ab-fiber
```
